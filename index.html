System Instrukcji IT i Utrzymania Ruchu

Kliknij na plik, aby wyświetlić instrukcje

PCWinSafe-J (Toyopuc Safety)
▼

Instrukcje IT

  • File .bat działa przez funkcje Windows Powershell za pomocą (runas /savecred) co potrzebuje przy pierwszym uruchomieniu hasło (od użytkownika Admin) a potem już zapamiętuje jego na zawsze.

Instrukcje Utrzymania Ruchu

  • Żeby Plik PCWinSafe-J pracował z Uprawnieniami Administratora należy uruchomić plik .bat "C:\Users\LocalUser\Desktop\PCwinSafeJ.bat"
